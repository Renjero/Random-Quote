# ✨ Random Quote Generator

A modern and responsive **Random Quote Generator** built with **HTML, Tailwind CSS, and JavaScript**. Generate inspiring quotes instantly with a clean, elegant, and user-friendly interface.

---

## 🚀 Live Demo

🔗 **View Project:**
https://random-quote-seven-lake.vercel.app

---

## ✨ Features

* 🎲 Generate random quotes instantly
* 📱 Fully responsive design
* 🎨 Modern UI with Tailwind CSS
* ⚡ Fast and lightweight
* 💻 Beginner-friendly JavaScript project
* 🖱️ Clean and interactive user experience
